### Comparable vs Comparator
Comparable: A comparable object is capable of comparing itself with another object. The class itself must implements the java.lang.Comparable interface in order to be able to compare its instances.

Comparator: A comparator object is capable of comparing two different objects. The class is not comparing its instances, but some other classâ€™s instances. This comparator class must implement the java.util.Comparator interface.
